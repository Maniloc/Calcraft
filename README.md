# CalCraft — Годовой Календарь

Конструктор годовых настенных календарей. Добавь фото, настрой дизайн, скачай PDF.

## Возможности

- 📅 Годовой календарь — все 12 месяцев на одном листе
- 🖼 Фото-обложка с кроппером (выбор области)
- 🎨 6 тем + настраиваемый акцентный цвет
- 📐 Экспорт A2 / A3 / A4 / Квадрат
- 🗂 Раскладка 3×4, 4×3 или 2×6
- 📅 Любые выходные дни
- 🎉 События и праздники с повторением каждый год
- ⬇ Экспорт PNG и PDF в 2× качестве

## Стек

Vanilla JS (ES Modules) · html2canvas · jsPDF · Google Fonts

## Структура

```
calcraft/
├── index.html
├── css/style.css
├── js/
│   ├── state.js   — состояние
│   ├── render.js  — отрисовка
│   ├── crop.js    — кроппер
│   ├── export.js  — экспорт
│   └── main.js    — точка входа
└── README.md
```

## Запуск локально

```bash
npx serve .
# или
python3 -m http.server 8080
```

> ES Modules требуют HTTP-сервер, file:// не работает.

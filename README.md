# CalCraft — Конструктор Календарей

Веб-приложение для создания кастомных календарей. Открыл — настроил — скачал.

## Возможности

- 📅 Выбор месяца, заголовок, подзаголовок
- 🎨 6 тем оформления + настраиваемый акцентный цвет
- 📐 4 формата экспорта (A4/A3 портрет/альбом, квадрат)
- 🖼 Обложка из изображения с кроппером и настройкой высоты
- 📅 Произвольные выходные дни
- 🏷 Мероприятия с цветовыми метками
- 🎉 Праздники
- 📊 Легенда
- ⬇ Экспорт в PNG и PDF (2× quality)

## Стек

- Vanilla JS (ES Modules) — без фреймворков
- [html2canvas](https://html2canvas.hertzen.com/) — рендеринг в canvas
- [jsPDF](https://github.com/parallax/jsPDF) — генерация PDF
- Шрифты: DM Serif Display + DM Sans (Google Fonts)
- Хостинг: GitHub Pages (статика)

## Структура

```
calcraft/
├── index.html
├── css/
│   └── style.css       — дизайн-система, темы, компоненты
├── js/
│   ├── state.js        — состояние приложения и константы
│   ├── render.js       — отрисовка DOM
│   ├── crop.js         — модальное окно кропа изображения
│   ├── export.js       — экспорт PNG / PDF
│   └── main.js         — точка входа, привязка событий
└── README.md
```

## Запуск локально

```bash
# Нужен любой статический сервер (из-за ES Modules)
npx serve .
# или
python3 -m http.server 8080
```

Затем открой http://localhost:8080

> ⚠️ Открытие `index.html` напрямую (file://) не работает из-за ES Modules — нужен HTTP-сервер.

## Деплой на GitHub Pages

1. Создай репозиторий на GitHub
2. Загрузи все файлы
3. В настройках репозитория → Pages → выбери ветку `main`, папку `/root`
4. Сайт будет доступен по адресу `https://<username>.github.io/<repo>`
